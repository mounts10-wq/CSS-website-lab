# Personal Portfolio Website - CSS Web Project

## Overview
A responsive personal portfolio website built with HTML and CSS. This project showcases a developer's skills, background transition from welding to software engineering, and completed projects.

## Project Contents
- `index.html` - Main website markup with semantic HTML structure
- `style.css` - Complete styling with responsive design

## Features
- **Header**: Centered section with name, tagline, and GitHub profile link
- **About Me**: Biography highlighting career transition and professional background
- **Skills Section**: Lists programming languages (C, Python, JavaScript, HTML)
- **Projects Grid**: Responsive grid showcasing 2 projects
  - JavaScript Calculator with History
  - HTML Travel Blog
- **Responsive Design**: Mobile-friendly layout using CSS Grid and Flexbox
- **Modern Styling**: Purple gradient theme with hover effects

## How to Use
1. Open `index.html` in a web browser
2. Explore all sections by scrolling
3. Click "GitHub Profile" link to visit developer's GitHub

## CSS Highlights
- Gradient background on header
- CSS Grid for responsive projects layout
- Flexbox for navigation and project tags
- Mobile breakpoints for tablets and phones
- Smooth transitions and hover effects

## Customization
- Update name in header (`<h1 class="name">`)
- Replace GitHub URL in navigation link
- Add or remove project cards in the projects grid
- Modify color scheme in CSS variables or hex colors
- Update skills list in the About Me section

## Browser Support
- Modern browsers (Chrome, Firefox, Safari, Edge)
- Mobile responsive for all screen sizes
